## 1️⃣ 栈的基础回顾

### 🍳 用生活例子理解栈

> 栈就像一摞盘子 —— 你每次只能从**最上面**拿走盘子（**后进先出**，LIFO）。

```
   ← 入栈 push          ← 出栈 pop
        | 5 |                    | 5 | →
        | 3 |                    | 3 |
        | 1 |                    | 1 |
        +---+                    +---+
```

Python 中，用**列表**就能当栈用：

```python
stack = []
stack.append(1)   # push → [1]
stack.append(2)   # push → [1, 2]
stack.append(3)   # push → [1, 2, 3]
top = stack.pop() # pop  → 3, stack = [1, 2]
```

> 🔑 栈就两个核心操作：**入栈（push）** 和 **出栈（pop）**。所有栈的进阶用法都是这两个操作的组合。

### 小白问：栈和队列有什么区别？🤔

**答**：栈是**后进先出（LIFO）** —— 最后放进去的最先出来，就像一摞盘子。队列是**先进先出（FIFO）** —— 最先放进去的最先出来，就像排队买奶茶 🧋。记法：栈 = 自动步枪弹夹（最后一颗上膛的子弹最先打出），队列 = 奶茶店排队（先到的先喝）。

## 2️⃣ 单调栈 · 核心套路

### 🤔 什么是单调栈？

> 单调栈就是栈里的元素**保持单调性** —— 要么从栈底到栈顶一直递增（单调递增栈），要么一直递减（单调递减栈）。

**为什么要保持单调？** 因为当新元素破坏了单调性，说明"时机到了"—— 可以结算旧元素的结果了！

### 🎯 核心套路一：单调递减栈（求下一个更大元素）

**识别特征**：题目中出现"下一个更大的元素"、"下一个更高的温度"、"右边第一个比它大的"等关键词。

**模板代码**：

```python
def monotonic_decreasing(nums: list[int]) -> list[int]:
    """
    单调递减栈：找每个元素右边第一个比它大的元素
    栈内存下标，栈顶到栈底温度递减（值递增）
    """
    n = len(nums)
    res = [-1] * n       # -1 表示没找到
    stack = []           # 存下标
    
    for i in range(n):
        # 新元素 nums[i] 比栈顶大 → 破坏单调性，出栈结算
        while stack and nums[i] > nums[stack[-1]]:
            idx = stack.pop()
            res[idx] = nums[i]   # nums[i] 是栈顶右边第一个更大的
        stack.append(i)          # 当前下标入栈
    
    return res

# 示例
print(monotonic_decreasing([73, 74, 75, 71, 69, 72, 76, 73]))
# 输出: [74, 75, 76, 72, 72, 76, -1, -1]
```

**图解**（每日温度）：

```
温度: 73  74  75  71  69  72  76  73
                              ↑
                             当前 i=6, 温度=76

栈内下标: [3, 4, 5]  → 对应温度 [71, 69, 72]
                                            ↓ 76 > 72, 出栈!
栈内下标: [3, 4]     → 对应温度 [71, 69]
                                            ↓ 76 > 69, 出栈!
栈内下标: [3]        → 对应温度 [71]
                                            ↓ 76 > 71, 出栈!
栈内下标: []         → 空
                ↓ 76 入栈，栈 = [6]
```

### 🎯 核心套路二：单调递增栈（求面积/区间）

**识别特征**：题目中出现"最大矩形面积"、"区间最小值乘区间长度"、"两端边界确定面积"等关键词。

**模板代码**：

```python
def monotonic_increasing(heights: list[int]) -> int:
    """
    单调递增栈：以每个高度为矩形的高，计算能形成的最大面积
    栈内存下标，栈内高度单调递增
    """
    stack = []
    max_area = 0
    # 末尾补 0 保证最后清空所有栈内元素
    heights = heights + [0]
    
    for i in range(len(heights)):
        # 新柱子比栈顶矮 → 栈顶高度的右边界找到了，出栈计算面积
        while stack and heights[i] < heights[stack[-1]]:
            h = heights[stack.pop()]          # 以出栈高度为矩形的高
            left = stack[-1] if stack else -1  # 左边界：栈顶下标（前一个更矮的）
            w = i - left - 1                   # 右边界：当前下标
            max_area = max(max_area, h * w)
        stack.append(i)
    
    return max_area

# 示例
print(monotonic_increasing([2, 1, 5, 6, 2, 3]))  # 输出: 10
```

**图解**（柱状图中最大矩形）：

```
高度: [2, 1, 5, 6, 2, 3, 0]
                      ↑
                      i=4, 高度=2

栈内: [2, 3] → 高度 [5, 6]
           ↓ 2 < 6! 弹出 6
  高=6, 左=-1, 右=4 → 面积=6*(4-(-1)-1)=6*2=12
                 ↓ 弹出 5
  高=5, 左=1, 右=4 → 面积=5*(4-1-1)=5*2=10
```

### 小白问：单调栈里存下标而不是值？为什么？🤔

**答**：存下标可以同时知道**元素的值**（通过下标访问数组）和**元素之间的距离**（`i - idx`），而存值只知道值不知道位置。在计算"隔了几天"、"面积有多大"这类问题时，距离和值同样重要！所以单调栈99%的情况下存的是**下标**。

## 3️⃣ 辅助双栈 · 字符串解码

### 🎯 核心套路三：双栈保存上下文

**识别特征**：嵌套结构 `k[...]`，需要记住外层的信息，内层处理完后再拼回外层。

**模板代码**：

```python
def decode_string(s: str) -> str:
    """
    双栈解码：数字栈 + 字符串栈
    遇到 '[' 保存当前上下文，遇到 ']' 恢复并重复
    """
    num_stack = []   # 存倍数
    str_stack = []   # 存之前的字符串
    cur_num = 0
    cur_str = ''
    
    for ch in s:
        if ch.isdigit():                    # 数字：累加
            cur_num = cur_num * 10 + int(ch)
        elif ch == '[':                     # 左括号：保存上下文，重置
            num_stack.append(cur_num)
            str_stack.append(cur_str)
            cur_num, cur_str = 0, ''
        elif ch == ']':                     # 右括号：恢复上下文，拼接
            cur_str = str_stack.pop() + cur_str * num_stack.pop()
        else:                               # 字母：追加到当前字符串
            cur_str += ch
    
    return cur_str
```

**执行过程图解**（`3[a2[c]]`）：

```
初始: cur_num=0, cur_str=''

① 遇到 '3' → cur_num=3
② 遇到 '[' → num=[3], str=[''], cur_num=0, cur_str=''
③ 遇到 'a' → cur_str='a'
④ 遇到 '2' → cur_num=2
⑤ 遇到 '[' → num=[3,2], str=['', 'a'], cur_num=0, cur_str=''
⑥ 遇到 'c' → cur_str='c'
⑦ 遇到 ']' → cur_str='' + 'c'*2='cc', num=[3], str=['']
⑧ 遇到 ']' → cur_str='' + 'cc'*3='cccccc', 完成!
```

> 💡 **口诀**：遇到 `[` 存起来，遇到 `]` 倒出来，数字累加字母接。

## 4️⃣ 表达式计算 · 双栈与临时栈

### 🎯 核心套路四：表达式求值（含括号）

**识别特征**：题目中出现"计算表达式"、"基本计算器"、"含加减乘除和括号"等关键词。

**分类处理**：

| 类型 | 问题 | 方法 |
|------|------|------|
| 🟢 **仅加减+括号** | L224 基本计算器 | 一个栈存符号和结果，符号位 ±1 |
| 🟡 **加减乘除+无括号** | L227 基本计算器 II | 遇到 `*/` 立即计算，`+-` 压栈 |
| 🔴 **加减乘除+括号** | L772 基本计算器 III | 结合上两者 |

**模板代码一：仅加减+括号（L224）**

```python
def calculate_basic(s: str) -> int:
    """
    只有 + - 和括号
    用栈保存遇到 '(' 时的结果和符号
    """
    stack = []
    res, sign = 0, 1  # sign = 1 表示正，-1 表示负
    i = 0
    
    while i < len(s):
        ch = s[i]
        if ch.isdigit():
            num = 0
            while i < len(s) and s[i].isdigit():
                num = num * 10 + int(s[i])
                i += 1
            res += sign * num
            continue     # 已经处理了 i，跳过最后的 i += 1
        elif ch == '+':
            sign = 1
        elif ch == '-':
            sign = -1
        elif ch == '(':
            # 把当前结果和符号入栈，然后重置
            stack.append(res)
            stack.append(sign)
            res, sign = 0, 1
        elif ch == ')':
            # 弹出符号和之前的计算结果，拼起来
            sign = stack.pop()    # 之前存的符号
            prev = stack.pop()    # 之前的结果
            res = prev + sign * res
        i += 1
    
    return res
```

**模板代码二：加减乘除+无括号（L227）**

```python
def calculate_with_mul_div(s: str) -> int:
    """
    支持 + - * /，无括号
    + 和 -：数字带符号压栈
    * 和 /：栈顶出栈，与当前数字运算后重新入栈
    """
    stack = []
    num, op = 0, '+'
    
    for i, ch in enumerate(s):
        if ch.isdigit():
            num = num * 10 + int(ch)
        
        if ch in '+-*/' or i == len(s) - 1:
            if op == '+':
                stack.append(num)
            elif op == '-':
                stack.append(-num)
            elif op == '*':
                stack.append(stack.pop() * num)
            elif op == '/':
                # 注意：Python 的 // 对负数处理不同，用 int()
                stack.append(int(stack.pop() / num))
            op = ch
            num = 0
    
    return sum(stack)
```

> 💡 **口诀**：一栈存符号，二栈先乘除，括号要保存，数字累加读。

## 5️⃣ 高频题精讲

### 🏆 题1：解码字符串（L394 · 中等）

**题目描述**：给定编码字符串 `3[a]2[bc]`，返回解码后的 `aaabcbc`。编码规则：`k[encoded_string]`，方括号内字符串重复 k 次。

**解题思路**：🔑 遇到嵌套怎么办？用栈保存外层上下文！

```
遍历过程可视化（s = "3[a2[c]]"）:
  Position: char → 操作
  0: '3'    → cur_num = 3
  1: '['    → 保存(3,"")  ↑ 进入新层
  2: 'a'    → cur_str = "a"
  3: '2'    → cur_num = 2
  4: '['    → 保存(2,"a")  ↑ 再进一层
  5: 'c'    → cur_str = "c"
  6: ']'    → 弹出(2,"a") → "a" + "c"*2 = "acc"
  7: ']'    → 弹出(3,"")  → "" + "acc"*3 = "accaccacc" ✅
```

**代码**：

```python
def decodeString(s: str) -> str:
    num_stack = []
    str_stack = []
    cur_num = 0
    cur_str = ''
    
    for ch in s:
        if ch.isdigit():
            cur_num = cur_num * 10 + int(ch)
        elif ch == '[':
            num_stack.append(cur_num)
            str_stack.append(cur_str)
            cur_num, cur_str = 0, ''
        elif ch == ']':
            cur_str = str_stack.pop() + cur_str * num_stack.pop()
        else:
            cur_str += ch
    
    return cur_str
```

⏱ **时间复杂度**：O(n) | 🧠 **空间复杂度**：O(n)

### 🏆 题2：每日温度（L739 · 中等）

**题目描述**：给定温度数组 `[73,74,75,71,69,72,76,73]`，返回每个温度要等多少天才会出现更高的温度 → `[1,1,4,2,1,1,0,0]`。

**解题思路**：🔑 单调递减栈！从右到左？不，从左到右更自然 —— 遇到更大的就结算前面的。

**代码**：

```python
def dailyTemperatures(temperatures: list[int]) -> list[int]:
    n = len(temperatures)
    res = [0] * n
    stack = []  # 存下标，栈内温度递减
    
    for i in range(n):
        # 当前温度比栈顶温度高 → 栈顶找到了"下一个更高"
        while stack and temperatures[i] > temperatures[stack[-1]]:
            idx = stack.pop()
            res[idx] = i - idx   # 中间隔了几天
        stack.append(i)
    
    return res
```

⏱ **时间复杂度**：O(n) — 每个元素入栈出栈各一次！| 🧠 **空间复杂度**：O(n)

### 🏆 题3：柱状图中最大的矩形（L84 · 困难）

**题目描述**：给定柱状图高度 `[2,1,5,6,2,3]`，找到能勾勒出的最大矩形面积 → 10。

**解题思路**：🔑 单调递增栈！以每个柱子为高，找它能延伸的最远左右边界。

**关键技巧**：在数组末尾**补一个 0**，这样最后所有栈内元素都会被"强制出栈"结算。

**代码**：

```python
def largestRectangleArea(heights: list[int]) -> int:
    stack = []
    max_area = 0
    heights = heights + [0]  # 🔥 哨兵：强制清空栈
    
    for i in range(len(heights)):
        while stack and heights[i] < heights[stack[-1]]:
            h = heights[stack.pop()]          # 弹出高度作为矩形高
            # 左边界：栈顶（前一个更矮的柱子），没有则为 -1
            left = stack[-1] if stack else -1
            w = i - left - 1                   # 宽度
            max_area = max(max_area, h * w)
        stack.append(i)
    
    return max_area
```

⏱ **时间复杂度**：O(n) | 🧠 **空间复杂度**：O(n)

**小白问：为什么左边届是 stack[-1] 而不是 stack[-1]+1？🤔**

**答**：因为 `stack[-1]` 是弹出元素左边第一个**比它矮**的柱子，所以它真正的左边界是 `stack[-1] + 1`。计算宽度时：`w = i - (stack[-1] + 1) = i - stack[-1] - 1`。完美！✅

### 🏆 题4：最大矩形（L85 · 困难）

**题目描述**：给定由 '0' 和 '1' 组成的二维矩阵，找出只包含 '1' 的最大矩形面积。

**解题思路**：🔑 **分层处理**！把每一行当作"柱状图的底"，从底向上数连续的 '1' 就是"柱子高度"，然后调用 L84 题的函数。

```
矩阵:                      柱状高度:
1 0 1 0 0   第1行 →     [1, 0, 1, 0, 0]
1 0 1 1 1   第2行 →     [2, 0, 2, 1, 1]  ← 从顶向下数连续1
1 1 1 1 1   第3行 →     [3, 1, 3, 2, 2]
1 0 0 1 0   第4行 →     [4, 0, 0, 3, 0]
                          ↑ 找最大矩形
                    heights[2]=3, 左右延伸... 面积=6 ✅
```

**代码**：

```python
def maximalRectangle(matrix: list[list[str]]) -> int:
    if not matrix or not matrix[0]:
        return 0
    
    n = len(matrix[0])
    heights = [0] * n
    max_area = 0
    
    for row in matrix:
        # 更新柱状图高度
        for j in range(n):
            if row[j] == '1':
                heights[j] += 1
            else:
                heights[j] = 0  # 断了，归零！
        # 调用 L84 的函数
        max_area = max(max_area, largestRectangleArea(heights))
    
    return max_area
```

⏱ **时间复杂度**：O(m × n) — 每行 O(1) 更新高度 + L84 O(n) | 🧠 **空间复杂度**：O(n)

### 🏆 题5：基本计算器（L224 · 困难）

**题目描述**：计算字符串表达式 `(1+(4+5+2)-3)+(6+8)` 的值 → 23。只有加减和括号。

**解题思路**：🔑 遇到 `(` 把当前结果和符号存栈，重置继续解析括号内；遇到 `)` 从栈中恢复之前的上下文。

```python
def calculate(s: str) -> int:
    stack = []
    res = 0      # 当前累积的结果
    sign = 1     # 当前符号：+1 或 -1
    i = 0
    
    while i < len(s):
        ch = s[i]
        if ch.isdigit():
            num = 0
            while i < len(s) and s[i].isdigit():
                num = num * 10 + int(s[i])
                i += 1
            res += sign * num
            continue
        elif ch == '+':
            sign = 1
        elif ch == '-':
            sign = -1
        elif ch == '(':
            # 🚀 保存当前环境
            stack.append(res)
            stack.append(sign)
            res, sign = 0, 1  # 重置，开始新的括号内计算
        elif ch == ')':
            # 🔄 恢复环境：括号内结果 × 外部符号 + 外部结果
            sign = stack.pop()
            prev = stack.pop()
            res = prev + sign * res
        i += 1
    
    return res
```

⏱ **时间复杂度**：O(n) | 🧠 **空间复杂度**：O(n)

## 6️⃣ 性能优化与常见坑

### 🚨 常见错误 TOP 5

| # | 错误 | 正确做法 | 错误示例 |
|---|------|---------|---------|
| 1️⃣ | 忘记左右边界+1/-1 | 宽度 `i - stack[-1] - 1` | 直接 `i - stack[-1]` |
| 2️⃣ | 忘记补 0 哨兵 | heights 末尾加 0 强制清栈 | 漏了最后还在栈内的柱子 |
| 3️⃣ | 单调栈符号搞反 | "下一个更大"用递减栈 | 反过来用了递增栈 |
| 4️⃣ | 表达式计算中负号处理错 | `int(a/b)` 而非 `a//b` | `a//b` 在负数时向下取整 |
| 5️⃣ | 解码字符串中忘了 `cur_num` 累加 | `cur_num*10 + int(ch)` | 直接 `int(ch)` 只处理个位数 |

### ⚡ 性能要点

1. **单调栈是 O(n) 的神器**：每个元素入栈一次、出栈一次，整体 O(n)。比暴力 O(n²) 不知道高到哪里去了！
2. **哨兵值的妙用**：在数组末尾加一个最小/最大值，可以避免最后遍历栈，代码更简洁。
3. **`/` vs `//`**：
   - `int(a / b)` = 向零取整（C语言风格 ✅）
   - `a // b` = 向下取整（负数时不同 ❌）
   - 例：`int(-3/2) = -1`，但 `-3//2 = -2`

### 🎯 记忆口诀

```
单调栈啊单调栈，存下标不存值
```

## 7️⃣ OD机考实战指南

### 📊 本主题在OD中的出题特点

| 题型 | 出现频率 | 难度 | 推荐耗时 |
|------|---------|------|---------|
| 单调栈（下一个更大） | ⭐⭐⭐⭐ 高频 | 中等 | 15-20min |
| 单调栈（面积计算） | ⭐⭐⭐ 中频 | 困难 | 20-30min |
| 字符串解码 | ⭐⭐⭐ 中频 | 中等 | 15-20min |
| 表达式计算 | ⭐⭐ 低频 | 困难 | 25-35min |

### 💡 OD 100分 vs 200分

- **100分题**：一般是单调栈的**直接应用**，如"下一个更大元素"的变体，识别特征明显，套模板即可
- **200分题**：往往是 **单调栈 + 其他数据结构** 的组合题，比如单调栈 + 二维矩阵 / 单调栈 + 哈希表

### ⏱ 时间分配建议

```
遇到栈题 → 3分钟内判断类型：
  
  ┌─ 求"下一个更大/更小"？ → 单调栈模板 → 15min
  │
  ├─ 求"矩形面积/区间"？  → 单调递增栈 → 20min
  │
  ├─ 有 k[...] 嵌套？     → 双栈解码 → 15min
  │
  └─ 有 +-*/ 符号？      → 表达式计算栈 → 25min
```

### 小白问：OD机考中怎么快速判断用单调栈？🤔

**答**：看到"下一个更大的/更小的"、"右边第一个比它大的"、"柱状图/直方图最大面积"、"区间最小值乘长度"这些关键词，直接敲**单调栈模板**！模板背熟，20秒就能写出框架，剩下的就是微调细节。OD机考80%的栈题都是单调栈的变体！

## 8️⃣ 今日作业

### ⭐ 简单题

**题1**：**下一个更大元素 I**（L496）
> 给定两个没有重复元素的数组 nums1 和 nums2，nums1 是 nums2 的子集。找出 nums1 中每个元素在 nums2 中的下一个更大元素。
> 💡 提示：先用单调栈算出 nums2 所有元素的下一个更大值，存到哈希表里

### ⭐ 中等题

**题2**：**去除重复字母**（L316）
> 给你一个字符串 s，请你去除重复字母，使每个字母只出现一次。需保证返回结果的字典序最小（要求不能打乱其他字符的相对位置）。
> 💡 提示：单调栈 + 计数哈希 —— 栈内保持字典序递增

**题3**：**股票价格跨度**（L901）
> 设计一个算法，计算股票在当日价格的跨度。跨度定义为当日及之前，股票价格小于等于当日价格的最大连续天数。
> 💡 提示：单调递减栈，栈内存 (价格, 跨度)

### ⭐⭐⭐ 进阶题

**题4**：**接雨水**（L42）🔥 OD高频！
> 给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。
> 💡 提示：单调递减栈，遇到比栈顶高的，弹出栈顶计算积水面积。或者用双指针法（面试加分）

**题5**：**基本计算器 III**（L772 🔒）
> 实现一个基本计算器来计算简单表达式字符串。表达式字符串可以包含非负整数、运算符 `+`、`-`、`*`、`/`、左右括号和空格。
> 💡 提示：结合模板一（括号处理）+ 模板二（乘除优先），中缀转后缀也可以

> 🎉 **今日总结**：栈的三种进阶应用你都掌握了！
> - 单调栈 → 处理"下一个更大/更小"的一维问题，O(n) 解决 O(n²) 问题
> - 双栈解码 → 处理嵌套结构 `k[...]`，遇到 `[` 保存上下文
> - 表达式计算 → 括号处理用栈保存环境，乘除优先用栈先算
>
> **记住**：栈就是"暂时不用的数据先存起来"～ 遇到嵌套、依赖上下文、需要回溯的场景，先问问自己"能不能用栈？" 💪
